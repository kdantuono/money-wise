
# MoneyWise

**MoneyWise** is a modern web and mobile application designed to automate personal finance management. Unlike traditional budgeting tools, MoneyWise intelligently parses your emails and notifications to detect transactions, subscriptions, and income, providing users with a dynamic and frictionless financial overview.
